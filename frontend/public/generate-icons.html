🎨 Генератор иконок AirBorn

Нажмите на кнопки ниже, чтобы скачать иконки разных размеров.

📋 Инструкция:

  1. Нажмите "Скачать все иконки" или скачайте каждую по отдельности
  2. Переместите все файлы в папку frontend/public/
  3. Пересоберите Docker образ: docker compose build --no-cache frontend
  4. Для OG image создайте изображение 1200x630 с логотипом
